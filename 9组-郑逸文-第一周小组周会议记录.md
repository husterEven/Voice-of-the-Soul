## 一、现阶段项目进度

### 1.1 项目选题与立项

本周完成项目选题确认，确定项目名称为**"面向内容创作者的个性化有声读物智能生成系统"**。项目面向自媒体创作者、知识付费讲师、播客主播、教育内容生产者等用户群体，通过Web端提供"选择场景 → 粘贴文本 → 一键生成 → 快速导出"的极简有声内容制作流程。

### 1.2 系统整体框架确认

经小组讨论，系统划分为三大模块：

| 模块 | 名称 | 核心内容 |
|------|------|---------|
| M1 | 用户端-创作中心模块 | 场景选择、创作工作台（文本输入/音色选择/参数调节/背景音乐）、音频合成与导出、作品管理、个人中心 |
| M2 | AI服务层-核心智能模块 | 文本预处理引擎、语音合成引擎（Edge-TTS）、情感处理引擎、音频合成引擎 |
| M3 | 基础支撑模块 | 用户认证（JWT）、文件存储服务、异步任务队列（Redis+Celery）、缓存服务、日志与监控 |

### 1.3 技术选型初步确定

- **后端框架**：Python Flask / FastAPI（待最终确认）
- **TTS引擎**：首选Edge-TTS（微软免费），备选阿里云TTS
- **音频处理**：pydub + FFmpeg
- **异步任务**：Redis + Celery
- **文件存储**：本地存储 + 腾讯云COS（备选）
- **数据库**：MySQL + SQLAlchemy ORM
- **前端框架**：Vue.js / React（待定，倾向于Vue3）
